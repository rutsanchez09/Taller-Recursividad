# Taller de Recursividad - Estructuras de Datos 
**Rut Ester Sanchez Arroyo**

Este repositorio contiene la solución a los 14 ejercicios propuestos para el taller de Recursividad

##  Contenido del Taller
El proyecto está organizado en un paquete llamado `recursividad`, donde cada punto se encuentra implementado en una clase independiente para mayor orden y claridad.

Los ejercicios incluidos son:
1.  **Factorial** de un número.
2.  **Sumatoria** de los primeros N números.
3.  **Serie Armónica** (1 + 1/2 + 1/3...).
4.  **Inversión** de un número entero.
5.  **Suma de dígitos** de un número.
6.  **Cociente** mediante restas sucesivas.
7.  **Potencia** (Base elevada a Exponente).
8.  **Máximo Común Divisor** (Algoritmo de Euclides).
9.  **Multiplicación** mediante sumas sucesivas.
10. **Suma de elementos** de un vector.
11. **Suma de elementos** de una matriz (m x n).
12. **Serie de Fibonacci** hasta un límite.
13. **Función de Ackermann**.
